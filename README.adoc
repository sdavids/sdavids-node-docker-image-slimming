== Branch Content

This branch utilizes:

* the default variant of the https://hub.docker.com/_/node/[offical NodeJS Docker image] for building the image
* the default variant of the https://hub.docker.com/_/node/[offical NodeJS Docker image] for serving the example REST API
* no optimizations
